# 出库单模块

## 页面范围

- 管理出库单：查询、查看、编辑、审核、反审核、删除、回收站恢复、彻底删除、打印数据入口。
- **列表默认只显示已审核**（`pass='1'`）；打开「显示未审核」或点顶部「审核申请」才看待审核单。
- **列表分页**：默认 **10 条/页**；可选 **10 / 20 / 50 / 100 / 200**（后端上限 200）。
- **列表「出库单数据」列**（出库日期后）：两行汇总，已去掉单独的「数量 / 不含税金额 / 含税金额」列（方案 B）。
  - 第 1 行：`关联单位`（主表 `kehu`）、`货仓`（主表 `ck`）——数值**蓝色**。
  - 第 2 行：`总项数`（蓝）、`含税总价`（紫）、`不含税总价` / `税点总价` / `出库总数量`（深红）；无 `price` 权限时不显示三个价格。
  - 汇总口径：明细 `kcaq01=出库单号` 且 `del=0` 行数；`kcaq051`/`kcaq05`/`kcaq03` 合计；税点 = 含税 − 不含税差额合计（列表 SQL 已聚合）。
- **列表展开明细**（2026-06-24）：主列表最左展开箭头，点击后按需调用 `GET /api/stock-out/:id` 拉明细；也可点行展开（点操作按钮区不触发）。
  - 主列表与入库单一致，使用视口底部固定横向滚动条（`v-erp-list-h-scroll`）。
  - 查询：`UB_ERP_Stocks_out_list` 中 `kcaq01=当前出库单号` 且 `del=0`，按 `seq` 排序；SQL 仅用 `l.*` + `colorName`，**禁止**再 `AS kcaq03` 等同名列（否则接口数值变数组，界面会显示 `0.00` 或 `2,2`）。
  - 列：序号（1、2、3…）、材料编码/名称/规格、颜色、单位、数量、单价、单价（含税）、金额、金额（含税）、税点、末两列（见下）；价格列受 `price` 权限控制。
  - **数量/单价/金额**：展开明细统一 **两位小数**、不加千分位逗号（如 `3.99`，不出现 `3.399` 或 `3,399`）。
  - **颜色**：主显 `kcaa11`，并按 `kcaa01` 中 `/` 后颜色代码查 `UB_ERP_Stocks_colorcode.code` 得名称，拼接为 `TM(深啡色)` 格式。
  - **备注**：仅显示 `Describe`。
  - **成品出库（类型 6）**末三列标题改为：报关单号（`Reference`）、报关型号（`Describe`）、报关单价（`kcaq08`）；其他类型为 PO/PI、备注（**已去掉唯一码列**）。
  - 表底「小计」行：**「小计：」在数量列左侧（单位列）**；汇总数量、不含税/含税金额；单价列为金额÷数量的加权平均；成品出库的报关单价列小计留空。
- **列表筛选区两行布局**与入库单一致：第一行「供应商/外协商 + 出库类型」，第二行「关键词 + 查询 + 回收站/显示未审核开关 + 重置」；输入框宽度 240 / 160 / 420 px。
- 出库单添加：分为“出库单基础资料”和“出库单明细”两个区域，布局参考入库单。
- **明细工具栏**（2026-06-24）：仅「批量添加」「删除选定明细」「删除全部明细」三按钮，样式与入库单一致（`primary plain` / `danger plain`）；**已移除**关键字搜索框、「增加明细」及关联型提示灰字。
- **明细删除**（2026-06-24）：表最左为「选择」列，点「删除」标记为「已选择」；「删除选定明细」只移除已标记行；「删除全部明细」二次确认后清空；删行仅改当前页内存，保存后才落库。
- **编辑页明细列**（2026-06-24）：顺序为选择 → 序号 → 材料编码/名称/规格/颜色/单位（**只读**）→ 数量 → 单价/单价(含税)/税点/金额/金额(含税)（有 `price` 权限）→ **厂款号/PI号**（可编辑，落库 `UB_ERP_Stocks_out_list.Reference`）→ 备注（`Describe`）；**已去掉「来源明细」列**（`kcaq02` 仍由批量添加/选单带回，保存时写入来源明细键供审核反写）。厂款号/PI号与备注列宽约 30 字（`min-width` 420px）。明细表使用与主列表相同的视口底部横向滚动条（`v-erp-list-h-scroll`）。
- **基础资料行序**（2026-06-24）：①出库单号 ②出库日期 ③出库类型 ④关联单号 ⑤关联单位 ⑥仓库+纸质单号+预留单号（**强制同一行**，三框各 200px、不换行）⑦是否含税 ⑧备注。
- **关联单号按类型**（2026-06-24，2026-06-25 更新）：其他出库可手填（选填）；采购退货为只读 +「选择」+「清空」，点击「选择」弹出采购订单列表页，支持供应商和关键字筛选，选中后回填 `kcap04`、`kcap05/kehu`、`sourceSystemcodeId` 并清空明细（选单后供应商下拉立即显示「编码 名称」）；**外协领料**为只读 +「选择」+「清空」，弹出外协订单列表，选中后回填 `kcap04`（外协单号）、`kcap08`（PI，只读）、`kcap05/kehu`（外协商）、`sourceSystemcodeId` 并清空明细；外协订单弹窗只保留一个查询框，关键字只支持 PI 号、外协商、外协单号模糊搜索；其它关联型仍可手填且必填。
- **外协领料基础资料**（2026-06-24）：选类型 `2` 时弹窗「是否需要加工后外协？」——确定则显示「加工车间」下拉（落库 `cj/cjname`），取消则为普通外协领料；关联单位标签为「外协商」，选项格式 `编码,名称`；**PI号**只读（`kcap08`）；外协领料时隐藏纸质单号/预留单号行。
- **外协领料批量添加**（2026-06-24，2026-06-25 更新，2026-06-24 BOM 子料展开）：类型 `2` 选定外协单后点「批量添加」→ 独立新窗口 `/inventory/daily/stock-out-assist-issue-batch-window`；须已选仓库、是否含税、外协单号、外协商；PI 为空时按外协单号查询，PI 不为空时叠加 PI 号精确筛选。
  - **列表列**（2026-06-24 列修正）：已去掉 RMB 单价/含税/金额/税点五列，以及返厂编码、实际入库数量列。核心列：库存账存数、物料未审出库数、实际库存（均按**子料 kcaa01 + 当前仓库**，页面固定三位小数显示）；还需出库数量 = 来源需求（换算外协数量×单用量）− 本单号+本仓+子料已审出库 − 未审出库；默认可选数量 = min(实际库存, 还需出库)；外协数量列显示换算后 `ksum(wxak03)`；实际出库数量 = 来源已审出库；未审出库情况 = 数量/单数/单号摘要；是否存在转换数据按子料单位显示。
  - **子料展开（旧系统口径）**：外协明细 `kcaa01` 仅为「外发对应编码」；批量列表一行 = 一条**实际出库子料**。带 PI 时按 `sid=PI` + `pq=款号(Product)` 读 `UB_ERP_Bom_pi_cost`；**子料候选**以 `UB_ERP_Bom_000.systemcode` 下 `UB_ERP_Bom_parts` **直接子层**为准（`wxak02` 仅兜底）。单用量：`top_kcaa01=外发对应` 时按 pi_cost 汇总 `kcac04`；BAG 锚点 PI（如 PI-4171）则用 Bom_parts 的 `kcac04`；无子料才兜底外协明细本身。
  - **列表列**：「需要外协材料编码」列内多行展示：子料编码（红字）、外发对应、外协数量、单用量；外协数量整数不显示 `.00`（已去掉独立「外发对应」列）。
  - **PI 提示**：仅无展开或全部兜底 `self` 时提示；Bom 子层已成功展开不弹误导蓝条。
  - **需要外协材料编码列**：首行子料编码（红字）；下方依次为外发对应（青绿辅助色）、外协数量、单用量；名称规格颜色等取**子料**资料。
  - **可选判断**：实际库存 > 0 且还需出库 > 0 时显示黄色「选择」；库存不足或来源已出满不可选。
  - 选行去重键 `wxak02|子料kcaa01`；带回 `kcaq02←wxak02`、`kcaa01←子料`、`kcaq03←默认可选量`、`kcaq031←实际库存上限`、子料资料快照、PO/PI、备注（外协明细 Describe/info）。

## 关键业务规则

- 新增默认未审核，系统不支持自动审核。
- 出库单号 `kcap01` 在保存时生成，前端只展示，不允许手工改。
- **保存落库快照**（2026-06-24）：主表 `uid/uname/utruename/addtime` 写入当前登录人；明细 `kcaa02_en` 抄 BOM 英文名称，`uid/uname/utruename/addtime` 同样写入当前操作员（与入库单明细一致）。
- 明细不能任意手填物料，必须来自库存或来源单据选择；**不再提供「增加明细」手加空行**。
- 未审核出库占用可用库存；审核后变成正式出库；反审核后重新回到未审核占用。
- 已结案旧数据仍禁止编辑、审核、反审核、删除（操作列可能显示「只读」）；界面状态列不单独展示结案标签。
- 审核/反审核会同步来源明细已出数量：
  - 采购退货：`UB_ERP_Buy_order_list.kcak07`
  - 外协领料/外协退货：`UB_ERP_assist_order_list.wxak08`
  - 生产领料：`UB_ERP_Dispatch_order_list.scak04`
  - 生产返修：`UB_ERP_Dispatch_order_list.scak05`
  - 成品出库：`UB_ERP_Sales_order_list.xsak06`

## SQL 兼容约束

本模块 SQL 必须兼容 SQL Server 2008 R2：

- 分页使用 `ROW_NUMBER()`。
- 不使用 `OFFSET`、`TRY_CONVERT`、`FORMAT`、`IIF`、`CONCAT`、`SEQUENCE`。
