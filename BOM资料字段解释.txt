- 物理表：bom_000（环境变量 INV_BOM_MASTER_TABLE 可覆盖表名，默认 bom_000）
- 本文件为界面 Label、接口字段与库列的逐字对应依据；未列出的列请勿在「BOM 基础资料」只读区随意展示。
- 以下「界面 Label」与前端表单项、弹窗表头文案保持一致。

【单位转换与转换率（与 Bom_unit_change 主数据关联）】
- 采购 ↔ 使用：由业务在维护界面选择「转换方式」下拉（默认：采购 → 使用；或 使用 → 采购）。具体存表列名以实现为准。
- 报价 ↔ 使用：同理（默认：报价 → 使用；或 使用 → 报价）。具体存表列名以实现为准。
- 转换率：不硬编码在说明文件；由「单位转换率」表（Bom_unit_change：unit_name、unit_name_tow、change_bl 等）按当前主档单位组合自动带出，只读展示。未匹配到记录时显示空或「—」。

【业务标记（多为 0/1 或 Y/N，以库为准；界面用禁用复选框只读）】
kcaa12	采购	勾选=启用该业务方向（与库内约定真值一致即可展示为勾选）
kcaa13	外协	同上
kcaa14	自产	同上
kcaa15	生产车间	同上
Customer_supply	客供	1=勾选（是），2=不勾选（否）；若库为其它枚举，以实际字典为准，但接口应对前端明确 true/false 或 1/2

【系统/审计类（一般不在「基础资料」Tab 首屏强塞，需要时可另 Tab 或折叠区）】
- pass：审核（常见 0 未审 / 1 已审）
- del：逻辑删除（在册多约定 0 或空为在册）
- sign：状态/标记（与库存模块约定一致）
- version：版本
- addtime / edittime：录入时间 / 修改时间（常为 nvarchar 业务时间串）

【明确不使用】
- kcaa16：当前 BOM 资料查看弹窗「基础资料」步骤不查询、不展示（剔除）。
