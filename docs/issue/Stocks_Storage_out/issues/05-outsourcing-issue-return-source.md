# 05 - 外协领料与外协退货来源选择

**Status:** `ready-for-agent`  
**Type:** AFK

## Parent

`docs/issue/Stocks_Storage_out/PRD.md`

## What to build

交付 `2` 外协领料和 `3` 外协退货的来源选择、带料和保存校验：用户必须选择外协单和外协客户，从外协相关来源中选择明细，保存时严控来源数量和库存可用数量。

第一版继续兼容旧系统 `wxak08` 作为外协相关已出数量累计字段，不拆分“已领”和“已退”两个字段。本工单不做审核反写；审核反写在后续生命周期工单中完成。

## Acceptance criteria

- [ ] `2` 外协领料和 `3` 外协退货新增时必须选择外协单号 `kcap04` 和外协客户。
- [ ] 来源外协单必须未删除、已审核、未结案，且外协客户与当前出库单一致。
- [ ] 来源选择按当前仓库校验库存可用数量。
- [ ] 来源选择显示外协来源、物料编码、名称、规格、颜色、单位、库存可用数、还需出库数量、单价、金额、税点、PO/PI、备注等关键字段。
- [ ] 选中后带回外协来源明细标识、物料快照、数量、价格、金额、税点、PO/PI、备注。
- [ ] 保存时服务端重新校验外协来源和库存可用数量。
- [ ] 保存时禁止超过来源剩余数量或库存可用数量。
- [ ] 编辑当前单时，来源占用汇总排除当前出库单自身明细。
- [ ] 文档或代码注释明确 `wxak08` 第一版语义为外协相关已出数量累计。
- [ ] 旧系统 PI 相关外部选择页源码不完整的复杂入口不在本工单实现范围内。
