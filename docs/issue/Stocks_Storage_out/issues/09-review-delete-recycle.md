# 09 - 复核、删除、回收站与彻底删除

**Status:** `ready-for-agent`  
**Type:** AFK

## Parent

`docs/issue/Stocks_Storage_out/PRD.md`

## What to build

交付出库单审核后的复核锁单能力，以及待审核单据的软删除、回收站恢复和超级管理员彻底删除能力。复核后的出库单必须锁定，禁止编辑、反审核和删除。

## Acceptance criteria

- [ ] 只有 `pass=1`、`del=0`、未复核、`closed=0` 且类型不是 `7`、`8`、`10` 的出库单可复核。
- [ ] 复核后单据显示为已复核。
- [ ] 已复核出库单禁止编辑、反审核、删除和彻底删除。
- [ ] 只有 `pass=0`、`del=0`、未复核、`closed=0` 且类型不是 `7`、`8`、`10` 的出库单可软删除。
- [ ] 软删除后 `del=1`，写入删除人和删除时间。
- [ ] 软删除待审核出库单释放未审核库存占用。
- [ ] 回收站只显示已软删除的待审核出库单。
- [ ] 回收站可恢复待审核软删除出库单。
- [ ] 恢复后重新占用可用库存；如果库存不足，应阻止恢复并提示。
- [ ] 彻底删除仅超级管理员可操作，不仅靠 `delete` 权限开放。
- [ ] 彻底删除仅允许待审核且已删除的出库单。
- [ ] 复核、删除、恢复、彻底删除都写入操作日志。
