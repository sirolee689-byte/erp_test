# 03 - 编辑、金额计算与物料快照

**Status:** `ready-for-agent`  
**Type:** AFK

## Parent

`docs/issue/Stocks_Storage_out/PRD.md`

## What to build

交付待审核出库单的编辑保存路径：用户可以编辑待审核且未复核的出库单，明细按页面当前内容整批重写，保存时重新计算库存占用并排除当前单自身旧明细。

本工单同时交付出库单金额计算和物料快照保存口径，确保数量、价格、备注、来源标识等业务输入不被物料主档覆盖。

## Acceptance criteria

- [ ] 只有 `pass=0`、`del=0`、未复核、`closed=0` 且类型不是 `7`、`8`、`10` 的出库单可编辑。
- [ ] 已审核、已复核、已结案、已删除和不可开放类型只能查看，不能保存编辑。
- [ ] 编辑保存明细采用整批重写，不做行级差异更新。
- [ ] 编辑保存时库存可用数量计算排除当前出库单自身旧明细。
- [ ] 编辑保存后未审核出库占用以最新明细为准。
- [ ] 金额计算以不含税单价与税点为主要输入，计算含税单价、不含税金额、含税金额。
- [ ] 用户直接修改含税单价时，页面反算不含税单价。
- [ ] `in_tax=2` 不含税模式时税点只能为 `0`。
- [ ] 服务端按物料编码重查物料主档并写快照。
- [ ] 数量、价格、备注、来源明细标识等业务输入字段不被物料主档覆盖。
- [ ] 历史旧数据查看不因物料已删除而报错。
