# 07 - 成品出库销售来源选择

**Status:** `ready-for-agent`  
**Type:** AFK

## Parent

`docs/issue/Stocks_Storage_out/PRD.md`

## What to build

交付 `6` 成品出库的销售订单来源选择、带料和保存校验：用户必须选择销售订单和客户，从销售订单相关来源中选择成品明细，保存时不能超过销售应出数量减已出数量再减未审核占用数量，也不能超过当前库存可用数量。

本工单只做到待审核保存和编辑校验，不做审核反写；审核反写在后续生命周期工单中完成。

## Acceptance criteria

- [ ] `6` 成品出库新增时必须选择销售订单单号 `kcap04` 和客户。
- [ ] 来源销售订单必须未删除、已审核、未结案，且客户与当前出库单一致。
- [ ] 来源明细来自销售订单明细。
- [ ] 可出数量按销售应出数量减已出数量再减未审核占用数量计算。
- [ ] 最终可出数量不得超过当前仓库同库存维度的可用库存。
- [ ] 选中后带回销售来源明细标识、成品物料快照、数量、价格、金额、税点、PI/PO 或备注等可用字段。
- [ ] 保存时服务端重新校验销售来源和库存可用数量。
- [ ] 保存时禁止超过来源剩余数量或库存可用数量。
- [ ] 编辑当前单时，来源占用汇总排除当前出库单自身明细。
- [ ] 类型 `10` 销售出库不在本工单开放新增或编辑，避免和 `6` 成品出库混淆。
