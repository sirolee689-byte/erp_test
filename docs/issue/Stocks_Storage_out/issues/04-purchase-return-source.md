# 04 - 采购退货来源选择与保存校验

**Status:** `ready-for-agent`  
**Type:** AFK

## Parent

`docs/issue/Stocks_Storage_out/PRD.md`

## What to build

交付 `1` 采购退货的来源选择、带料和保存校验：用户必须选择采购单和供应商，从该采购单已入库且当前仓库可退的物料中选择明细，保存时不能超过来源可退数量或当前库存可用数量。

本工单只做到待审核保存和编辑校验，不做审核反写；审核反写在后续生命周期工单中完成。

## Acceptance criteria

- [ ] `1` 采购退货新增时必须选择采购单号 `kcap04` 和供应商。
- [ ] 来源采购单必须未删除、已审核、未结案，且供应商与当前出库单供应商一致。
- [ ] 来源明细来自采购单明细，并结合已审核入库、已审核出库、未审核出库占用计算可退数量。
- [ ] 最终可退数量不得超过当前仓库同库存维度的可用库存。
- [ ] 来源选择页显示材料编码、名称、规格、颜色、单位、库存可用数、可退数量、单价、金额、税点、PO/PI、备注等关键字段。
- [ ] 选中后带回物料快照、来源明细标识、数量、价格、金额、税点、PO/PI、备注。
- [ ] 保存时服务端重新校验采购来源和库存可用数量。
- [ ] 保存时禁止超过来源可退数量或库存可用数量。
- [ ] 保存时保留采购退货后续审核需要的来源明细标识。
- [ ] 编辑当前采购退货单时，来源占用汇总排除当前出库单自身明细。
