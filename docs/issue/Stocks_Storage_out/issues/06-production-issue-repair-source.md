# 06 - 生产领料与生产返修来源选择

**Status:** `ready-for-agent`  
**Type:** AFK

## Parent

`docs/issue/Stocks_Storage_out/PRD.md`

## What to build

交付 `4` 生产领料和 `5` 生产返修的来源选择、带料和保存校验：用户按生产车间、派工或 PI 生产来源选择明细，保存时不能超过来源应出数量减已出数量再减未审核占用数量，也不能超过当前库存可用数量。

本工单只做到待审核保存和编辑校验，不做审核反写；审核反写在后续生命周期工单中完成。

## Acceptance criteria

- [ ] `4` 生产领料必须按已定业务要求选择派工单号、PI、生产车间和仓库。
- [ ] `5` 生产返修必须选择生产来源，至少满足生产返修第一版已确认的派工来源要求。
- [ ] 来源派工或生产单据必须未删除、已审核、未结案，且生产车间与当前出库单一致。
- [ ] 来源明细来自生产或派工相关明细。
- [ ] 生产领料可出数量按来源应领数量减已领数量再减未审核占用数量计算。
- [ ] 生产返修可出数量按来源应返修数量减已返修数量再减未审核占用数量计算。
- [ ] 保存时服务端重新校验生产来源和库存可用数量。
- [ ] 保存时禁止超过来源剩余数量或库存可用数量。
- [ ] 生产领料和生产返修保存时保留后续审核反写需要的来源明细标识。
- [ ] 编辑当前单时，来源占用汇总排除当前出库单自身明细。
