# 11 - 不开放类型、结案旧数据与边界占位

**Status:** `ready-for-agent`  
**Type:** AFK

## Parent

`docs/issue/Stocks_Storage_out/PRD.md`

## What to build

交付第一版不开放范围的保护：类型 `7` 生产领料（计划外）、`8` 生产领料（补数）、`10` 销售出库只读展示，不开放新增、编辑、审核、反审核、复核、删除；`closed=1` 历史旧数据只读展示，不提供结案和反结案入口。

## Acceptance criteria

- [ ] 新增和编辑表单不能选择类型 `7`、`8`、`10` 创建新单。
- [ ] 历史类型 `7`、`8`、`10` 可在列表和详情查看。
- [ ] 历史类型 `7`、`8`、`10` 显示不可操作标志。
- [ ] 历史类型 `7`、`8`、`10` 禁止编辑、审核、反审核、复核、删除、恢复和彻底删除。
- [ ] 第一版不显示结案和反结案入口。
- [ ] 新建出库单默认 `closed=0`。
- [ ] `closed=1` 历史出库单可查看但禁止编辑、审核、反审核、复核、删除、恢复和彻底删除。
- [ ] 导出按钮只作为待开发入口，不生成真实 Excel。
- [ ] 页面文案明确这些功能暂不开放，避免用户误以为系统故障。
