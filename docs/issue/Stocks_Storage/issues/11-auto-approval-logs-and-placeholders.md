# 11 - 自动审核、操作日志与待开发占位

**Status:** `ready-for-agent`  
**Type:** AFK

## Parent

`docs/issue/Stocks_Storage/PRD.md`

## What to build

交付第一版的自动审核能力和所有待开发占位：当旧系统已有用户配置允许自动审核时，新增保存直接成为已审核入库单并进入库存统计；否则保存为待审核。同时补齐关键操作日志和“导出”“超量入库配置”的待开发入口。

本工单不开发新的自动审核配置页面，不开发真实 Excel 导出，不开发超量豁免规则。

## Acceptance criteria

- [ ] 新增保存时读取既有用户自动审核配置字段。
- [ ] 未满足自动审核配置时，保存结果为 `pass=0`。
- [ ] 满足自动审核配置时，保存结果为 `pass=1`，并同步明细审核状态。
- [ ] 自动审核时审核人写当前保存人，审核时间写保存成功时间。
- [ ] 自动审核后的入库单立即进入库存统计口径。
- [ ] 操作日志能区分普通保存待审和保存并自动审核。
- [ ] 操作日志覆盖新增、编辑、审核、反审核、软删除、恢复、彻底删除。
- [ ] 列表提供“导出”待开发入口，但不生成真实 Excel。
- [ ] 页面或菜单提供“超量入库配置”待开发占位。
- [ ] 超量豁免第一版不生效，不硬编码特殊供应商。
