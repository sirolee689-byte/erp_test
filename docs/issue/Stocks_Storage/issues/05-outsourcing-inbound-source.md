# 05 - 外协入库与外协退料关联外协单

**Status:** `ready-for-agent`  
**Type:** AFK

## Parent

`docs/issue/Stocks_Storage/PRD.md`

## What to build

交付外协入库和外协退料的端到端路径：用户选择外协客户和外协单，从外协单批量带入可入库或可退料明细，保存成待审核入库单。

本工单沿用采购入库已经打通的关联来源模式，但接入外协订单主从表和外协客户匹配规则。第一版只读取和校验外协单，不反写外协单数量。

## Acceptance criteria

- [ ] 新增页支持入库类型 `2` 外协入库和 `3` 外协退料。
- [ ] 外协入库和外协退料必须选择外协客户和外协单。
- [ ] 服务端校验外协单已审核、未结案、未删除，并且外协客户与当前入库关联方一致。
- [ ] 外协客户名称保存时由服务端按编码重新查询并写入快照。
- [ ] 批量添加窗口按外协单展示可选明细。
- [ ] 批量添加带回材料快照、关联订单明细键 `kcao02`、关联单号 `kcan04` 和可入库数量上限。
- [ ] 外协入库和外协退料不允许手工新增无外协来源的物料行。
- [ ] 保存时后端再次校验外协单与外协客户关系。
- [ ] 保存时后端按实时可入库数量阻止超量。
- [ ] 保存成功后单据为待审核，库存不变化。
