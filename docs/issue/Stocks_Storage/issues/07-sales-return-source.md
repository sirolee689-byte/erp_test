# 07 - 销售退货关联销售订单

**Status:** `ready-for-agent`  
**Type:** AFK

## Parent

`docs/issue/Stocks_Storage/PRD.md`

## What to build

交付销售退货入库的端到端路径：用户选择客户和销售订单，从销售订单批量带入退货入库明细，保存成待审核入库单。

本工单只读取和校验销售订单，不反写销售订单退货数量。

## Acceptance criteria

- [ ] 新增页支持入库类型 `6` 销售退货。
- [ ] 销售退货必须选择客户和销售订单。
- [ ] 服务端校验销售订单已审核、未结案、未删除，并且客户与当前入库客户一致。
- [ ] 客户名称保存时由服务端按客户编码重新查询并写入快照。
- [ ] 批量添加窗口按销售订单展示可退货入库明细。
- [ ] 批量添加带回材料快照、关联订单明细键 `kcao02`、关联单号 `kcan04` 和可入库数量上限。
- [ ] 销售退货不允许手工新增无销售来源的物料行。
- [ ] 保存时后端再次校验销售订单与客户关系。
- [ ] 保存时后端按实时可入库数量阻止超量。
- [ ] 保存成功后单据为待审核，库存不变化。
