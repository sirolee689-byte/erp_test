# 10 - 价格权限与入库单打印

**Status:** `ready-for-agent`  
**Type:** AFK

## Parent

`docs/issue/Stocks_Storage/PRD.md`

## What to build

交付价格权限和打印闭环：有价格权限的用户能在页面和打印中看到单价、税点、金额；没有价格权限的用户在列表、详情、编辑、打印中都看不到价格字段，且后端不允许其覆盖价格字段。

同时实现入库单打印页，打印主表信息、明细和合计，不兼容旧 IE 页眉页脚设置。

## Acceptance criteria

- [ ] 权限动作 `price` 控制单价、税点、金额字段可见性。
- [ ] 无价格权限用户在列表、详情、编辑、批量添加结果、打印中看不到价格相关字段。
- [ ] 无价格权限用户提交保存时，服务端不能让其覆盖已有价格字段。
- [ ] 有价格权限用户可查看和编辑单价、税点、金额相关字段。
- [ ] 打印入口受 `view` 权限控制。
- [ ] 打印页包含入库单号、入库日期、类型、仓库、关联方、经手人、纸质单号、备注、明细、合计。
- [ ] 打印页价格字段受 `price` 权限控制。
- [ ] 打印使用现代 CSS 打印样式，不做 IE 页眉页脚兼容。
- [ ] 打印数据与只读详情一致。
- [ ] 打印相关操作或访问失败时有明确提示。
