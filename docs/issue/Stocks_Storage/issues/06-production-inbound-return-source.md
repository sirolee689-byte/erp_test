# 06 - 生产入库与生产退料关联派工单

**Status:** `ready-for-agent`  
**Type:** AFK

## Parent

`docs/issue/Stocks_Storage/PRD.md`

## What to build

交付生产入库和生产退料的端到端路径：生产入库必须关联派工单并从派工单带出明细；生产退料允许不填派工单，不填时可按仓库或手工选料录入。

本工单要落实一个关键定稿：类型 `4` 生产入库派工单不能为空，类型 `5` 生产退料可以为空。

## Acceptance criteria

- [ ] 新增页支持入库类型 `4` 生产入库和 `5` 生产退料。
- [ ] 生产入库必须选择生产车间和派工单。
- [ ] 生产入库保存时后端校验派工单已审核、未结案、未删除，并且生产车间匹配。
- [ ] 生产入库明细必须来自派工单，不允许手工新增无派工来源物料。
- [ ] 生产入库批量添加带回派工单明细、关联订单明细键 `kcao02`、关联单号 `kcan04` 和可入库数量上限。
- [ ] 生产入库保存时按实时可入库数量阻止超量。
- [ ] 生产退料允许派工单为空。
- [ ] 生产退料有派工单时按派工单带出和校验明细。
- [ ] 生产退料无派工单时允许按仓库筛选或手工选择物料，只校验数量大于 0。
- [ ] 生产车间名称保存时由服务端按编码重新查询并写入快照。
- [ ] 保存成功后单据为待审核，库存不变化。
