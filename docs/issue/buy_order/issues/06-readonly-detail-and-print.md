# 06 - 只读查看与采购单打印

**Status:** `ready-for-agent`  
**Type:** AFK

## Parent

`docs/issue/buy_order/PRD.md`

## What to build

交付采购单查看和打印闭环：用户可以从列表打开只读详情，查看基础资料、采购明细、额外费用和合计；有查看权限的用户可以打开专用打印视图，打印内容遵守价格权限。

本工单不做真实 Excel 导出，只保留未来导出入口的权限口径。

## Acceptance criteria

- [ ] 列表可打开采购单只读详情大弹窗。
- [ ] 只读详情展示基础资料、采购明细、额外费用、审核信息、结案信息和操作状态。
- [ ] 只读详情中所有字段不可编辑，不显示保存按钮。
- [ ] 有 `view` 权限的用户可以打开打印视图。
- [ ] 打印视图包含采购单头部、供应商、关联单号、币别、明细、额外费用、备注和合计。
- [ ] 有 `price` 权限时打印显示单价、税点、金额和合计。
- [ ] 无 `price` 权限时打印隐藏单价、税点、金额和合计，不能通过打印绕过价格权限。
- [ ] 打印动作写入操作日志。
- [ ] 导出入口第一版只做权限占位，不生成真实 Excel。

## Testing notes

- 建议覆盖只读详情不可编辑、打印数据包含费用、无 `price` 权限打印隐藏价格。
- 手工验证：分别用有价格权限和无价格权限账号打开同一张采购单打印，价格显示符合权限。
