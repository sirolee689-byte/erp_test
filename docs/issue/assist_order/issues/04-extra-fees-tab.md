# 04 - 额外费用清单页签

**Status:** `ready-for-agent`  
**Type:** AFK  

## Parent

`docs/issue/assist_order/PRD.md`

## What to build

交付外协订单第三个页签“额外费用清单”：用户可以录入运费、包装费等附加费用，费用编码及名称从 `bom_000` 费用资料选择，保存后费用参与订单汇总和打印数据。

新系统额外费用不限制条数，不沿用旧系统最多 10 条的硬编码限制。保存采用整批重写：先物理删除当前外协订单旧费用，再插入页面当前费用。

## Acceptance criteria

- [ ] 外协订单编辑窗口包含三个页签：基础资料、外协订单明细、额外费用清单。
- [ ] 额外费用页签可新增、删除、编辑费用行，未审核未结案时可操作，已审核或已结案时只读。
- [ ] 费用编码及名称从 `bom_000` 选择，过滤条件包含 `kcaa05='FEE'`，并只查已审核、未删除数据。
- [ ] 每行费用至少包含序号、费用编码及名称、费用金额、税点、备注。
- [ ] 费用金额参与订单汇总中的额外费用和总额展示。
- [ ] 费用行不限制 10 条，至少手工或自动验证超过 10 条可以保存和刷新。
- [ ] 保存费用采用整批重写，删除一行保存后不会残留旧费用。
- [ ] 费用保存、读取、列表详情、打印数据来源保持一致。
