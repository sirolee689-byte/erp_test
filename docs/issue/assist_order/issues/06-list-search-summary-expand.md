# 06 - 列表搜索、筛选、汇总与展开明细

**Status:** `ready-for-agent`  
**Type:** AFK  

## Parent

`docs/issue/assist_order/PRD.md`

## What to build

把外协订单列表从“能看”补齐到“能用”：支持单字段搜索、全字段模糊搜索、供应商筛选、外协类型筛选、结案状态筛选、未审核开关、回收站开关、排序、交货日期距今天数，并在列表中展示订单汇总和展开明细。

本工单不新增新的业务动作，只增强查询和展示。

## Acceptance criteria

- [ ] 默认查询为已审核、未删除。
- [ ] “显示未审核”开关可查询未审核单据。
- [ ] “回收站”开关可查询已删除单据。
- [ ] 结案状态筛选支持全部、未结案、已结案。
- [ ] 单字段搜索支持外协订单号 `wxaj01`、外协订单日期 `wxaj02`、关联单号 `wxaj04`、备注 `remark`。
- [ ] 全字段模糊搜索同时匹配外协订单号、日期、关联单号、外协商编码、外协商名称快照、币别名称快照、备注、打印注释。
- [ ] 外协商筛选支持从供应商下拉选择，并支持编码或名称模糊匹配。
- [ ] 外协类型筛选支持全部、订单外协、订单外发、其他外协。
- [ ] 排序支持按交货日期、外协日期、供应商。
- [ ] 列表显示交货日期距今天数，例如还剩 N 天、今天到期、逾期 N 天或空值。
- [ ] 列表汇总显示客户名/外协商名称、含税标记、总项数、总数量、含税总价、不含税总价、税点总价、额外费用。
- [ ] 展开行显示 PRD 约定的明细列，金额和数量格式与录入规则一致。
- [ ] 查询参数前后端命名清晰，刷新页面或重新查询不会丢掉当前筛选条件。
