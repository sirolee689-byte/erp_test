# 01 - 外协订单列表与只读详情

**Status:** `ready-for-agent`  
**Type:** AFK  

## Parent

`docs/issue/assist_order/PRD.md`

## What to build

交付外协订单模块的第一条可演示路径：用户能从菜单进入外协订单页面，看到真实列表，按默认规则分页查询已审核、未删除单据，并打开一张单据查看主表、明细和额外费用的只读详情。

本工单要打通前后端最小闭环：菜单/路由、列表接口、单笔详情接口、前端列表页、只读大弹窗或抽屉。详情里先只读展示三块内容：外协订单基础资料、外协订单明细、额外费用清单。

本工单不做新增、保存、审核、结案、删除、打印。

## Acceptance criteria

- [ ] 菜单进入外协订单页面时不再是占位页，页面标题和模块位置与供应链日常业务模块一致。
- [ ] 列表默认只查 `pass` 为已审核、`del` 为未删除的外协订单，并支持分页。
- [ ] 列表至少显示：操作、选择、结案状态、外协类型、外协订单号、外协日期、交货日期、关联单号、币别、外协商、备注、打印注释、审核状态。
- [ ] 单笔详情返回主表一行、明细数组、额外费用数组，前端可在同一个大弹窗或抽屉内查看。
- [ ] 明细只读列包含 `seq`、PI号、款号/材料编码、中文名、数量、不含税单价、含税单价、不含税金额、含税金额、税点、交货日期、参考单号、备注。
- [ ] 额外费用只读列包含序号、费用编码及名称、费用金额、税点、备注。
- [ ] 无数据、接口失败、未授权时有清晰提示，不出现空白页。
