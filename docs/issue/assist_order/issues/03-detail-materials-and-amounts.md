# 03 - 外协订单明细选材与金额联动

**Status:** `ready-for-agent`  
**Type:** AFK  

## Parent

`docs/issue/assist_order/PRD.md`

## What to build

交付外协订单明细录入闭环：用户可以在明细页签选择可外协物料，带入必要快照字段，录入数量、税点和单价，前端完成含税/不含税金额双向联动，保存后刷新不丢数据。

订单外协和订单外发优先从销售 BOM 选材，并按关联单号过滤，关联销售订单必须已审核。其他外协第一版只从 `bom_000` 选材，过滤已审核、未删除；旧系统采购报价来源 `UB_ERP_Buy_offer_list` 暂不做。

明细保存采用整批重写：先物理删除当前外协订单旧明细，再插入页面当前明细。明细允许重复物料，不自动去重或合并。

## Acceptance criteria

- [ ] 明细页签可新增、删除、编辑明细行，未审核未结案时可操作，已审核或已结案时只读。
- [ ] 订单外协、订单外发选材来源为 PI 销售 BOM，并按 `wxaj04` 过滤；只允许已审核销售订单来源。
- [ ] 其他外协第一版选材来源为 `bom_000`，只查已审核、未删除数据。
- [ ] 是否可选由物料档案上的可外协标记控制；可外协行显示橙色且可选，不可外协行显示灰色且不可选。
- [ ] 点击不可外协物料时提示“该物料未勾选外协”或同等清晰文案。
- [ ] 从 PI 销售 BOM 带入数量时，默认数量直接取销售订单明细行数量 `xsak03`，不做“BOM 单件用量乘销售订单数量”。
- [ ] 至少保存用于打印和对账的快照字段：物料编码、中文名、英文名、开票名、规格、单位、颜色、版本、产地、是否客户供应、来源 PI 号。
- [ ] 明细保存字段包含 `wxak03`、`wxak04`、`wxak041`、`tax`、`wxak05`、`wxak051`、`delivery_date`、`reference`、`wxak06`、`seq`。
- [ ] 用户输入 `tax=0.13` 表示 13%。
- [ ] 用户改不含税单价 `wxak04` 或税点 `tax` 时，自动计算含税单价、不含税金额、含税金额。
- [ ] 用户改含税单价 `wxak041` 时，按税点反算不含税单价，并刷新两套金额。
- [ ] 数量默认显示 2 位，金额固定 2 位，单价按主表 `decimal` 控制，默认 4 位。
- [ ] 后端保存金额字段时不重算，原样保存前端传入的五个金额/税点字段。
- [ ] 保存明细采用整批重写，删除一行保存后不会残留旧明细，重复物料保存后仍保留重复行。
