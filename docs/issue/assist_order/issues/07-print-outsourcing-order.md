# 07 - 外协单打印与批量连续打印

**Status:** `ready-for-agent`  
**Type:** AFK  

## Parent

`docs/issue/assist_order/PRD.md`

## What to build

交付第一版外协订单打印：固定使用旧系统“外协单”的外协格式 `wxgs=0`，不显示“外协内容”列。用户可单张打印，也可勾选多张订单连续打印，每张单后自动换页。

打印需要包含表头、明细、额外费用、合计、十二条固定合约条款和签字栏。每页行数和单价小数位按用户配置保存到 `UB_ERP_User_print_setup`。

## Acceptance criteria

- [ ] 打印第一版固定 `wxgs=0`，没有“外协内容/Describe”列，也没有格式切换开关。
- [ ] 打印表头实时从供应商表查询加工商简称 `s_sname`、结算方式、地址、联系人、联系电话。
- [ ] 打印表头显示日期、PI号/关联单号、币别名称快照、含税/不含税文字、备注。
- [ ] 明细列包含序号、材料编码、材料名称/规格、对应款号、配件颜色、组别、单位、数量、单价、金额、交期、税点。
- [ ] 配件颜色按 `Bom_colorcode` 查询颜色名称，并与颜色编码一起显示。
- [ ] 单价和金额按 `wxaj06` 决定：含税显示 `wxak041/wxak051`，不含税显示 `wxak04/wxak05`。
- [ ] 额外费用接在明细后继续排序号，金额列显示费用金额，其余列空白。
- [ ] 表尾合计包含数量合计和金额合计，金额合计按含税标记选择明细金额，并计入额外费用。
- [ ] 打印包含十二条固定外协加工合同条款。
- [ ] 签字栏包含甲方、应付会计、乙方、盖章、厂长、日期、制表人、核对栏；制表人显示当前登录用户真实姓名。
- [ ] 支持批量选择多张外协订单连续打印，每张单后自动换页。
- [ ] 每页行数按用户保存，默认 12 行，范围 3 到 15 行。
- [ ] 单价小数位按用户保存，默认 2 位，范围 2 到 5 位。
- [ ] 明细超出每页行数时自动分页，并在新页重复表头和订单信息。
